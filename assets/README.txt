ICON DATEI HIER ABLEGEN
=======================

Fuer den Build wird ein Icon benoetigt:

  icon.ico  (256x256 oder groesser, .ico Format)

Du kannst ein PNG zu ICO konvertieren mit:
  - https://convertio.co/png-ico/
  - https://icoconvert.com/

Oder ein eigenes Icon erstellen mit:
  - GIMP
  - Photoshop
  - Online Icon Generator
